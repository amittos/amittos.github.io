Projects

Here is a list of my projects and projects I have been a part of.

TweetFix
Crowd-sourcing in Social Media, including Twitter and Youtube, regarding match fixing and bet fixing as part of the ongoing Erasmus+ project FIX the FIXING: Proactive quelling of sports events manipulation (Collaborative Partnership). Tasks included:
  • Perform social media mining regarding match fixing cases
  • Enable automatic data visualization of the results
  • Build a tool for sentiment analysis of short informal texts
  • Develop a web platform for the presentation of the results
Refugee Crisis - A Twitter-based Social Analysis on Refugee Crisis. Our team performed Statistical Analysis, Topic Detection and Affective Analysis on data collected from Twitter, regarding the current worldwide Refugee Crisis.
BorderShare - This project was implemented during the BorderHack hackathon, the purpose of which was to help refugees. In the website, the citizens of Thessaloniki can look for places to give resources that will be sent to the refugees.
Haunted - A ruby game using the Gosu library. This game was developed based on the Learn Game Programming with Ruby book.
